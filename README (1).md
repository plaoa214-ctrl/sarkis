# 🏠 Прогнозирование стоимости домов с использованием нейронной сети

Проект для классификации домов по ценовым категориям на основе их характеристик с использованием глубокого обучения.

## 📋 Описание

Система машинного обучения, которая анализирует характеристики домов (площадь, количество комнат, этаж) и классифицирует их по трем ценовым категориям с помощью нейронной сети на базе TensorFlow/Keras.

## ✨ Особенности

- Автоматическая классификация домов по ценовым категориям
- Использование KBinsDiscretizer для равномерного распределения классов
- Нормализация признаков с помощью StandardScaler
- Нейронная сеть с архитектурой глубокого обучения
- Детальная оценка модели с помощью confusion matrix и classification report
- Валидация в процессе обучения

## 🛠 Технологии

- **Python 3.7+**
- **TensorFlow/Keras** - для построения и обучения нейронной сети
- **scikit-learn** - для предобработки данных и оценки модели
- **pandas** - для работы с данными
- **numpy** - для численных операций

## 🚀 Использование

### Базовое использование

Запустите основной скрипт:
```bash
python house_price_prediction.py
```

### Что делает скрипт:

1. **Генерация данных**: Создает синтетический датасет из 1000 домов с характеристиками:
   - Площадь: 20-500 м²
   - Количество комнат: 1-20
   - Этаж: 1-10
   - Цена: 500,000 - 50,000,000 рублей

2. **Предобработка**:
   - Разделение цен на 3 класса с помощью KBinsDiscretizer (quantile strategy)
   - Нормализация признаков (StandardScaler)
   - Разделение на обучающую (80%) и тестовую (20%) выборки

3. **Обучение модели**:
   - Архитектура: 32 → 16 → 3 нейрона
   - Функция активации: ReLU для скрытых слоев, Softmax для выходного
   - Оптимизатор: Adam
   - Loss: sparse_categorical_crossentropy
   - Валидация: 20% от обучающей выборки
   - Эпохи: 50
   - Batch size: 32

4. **Оценка модели**:
   - Confusion Matrix
   - Classification Report (precision, recall, f1-score)

## 📊 Архитектура модели

```
Входной слой (3 признака)
    ↓
Скрытый слой 1 (32 нейрона, ReLU)
    ↓
Скрытый слой 2 (16 нейронов, ReLU)
    ↓
Выходной слой (3 нейрона, Softmax)
    ↓
Классы: [0, 1, 2]
```

## 🔧 Параметры модели

Вы можете настроить следующие параметры в коде:

- `n_samples` - количество образцов в датасете (по умолчанию: 1000)
- `n_bins` - количество классов цен (по умолчанию: 3)
- `test_size` - размер тестовой выборки (по умолчанию: 0.2)
- `validation_split` - доля данных для валидации (по умолчанию: 0.2)
- `epochs` - количество эпох обучения (по умолчанию: 50)
- `batch_size` - размер батча (по умолчанию: 32)
- Количество нейронов в скрытых слоях (по умолчанию: 32, 16)
